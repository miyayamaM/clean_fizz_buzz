# 概要
SOLIDに則って拡張性の高い形で[FizzBuzz問題](https://e-words.jp/w/FizzBuzz%E5%95%8F%E9%A1%8C.html)を解く

[ちょうぜつソフトウェア設計入門](https://gihyo.jp/book/2022/978-4-297-13234-7)のコードのRust版

# モジュール

#### `factories`
インスタンスを生成する構造体

#### `outputs`
書き出しを行う構造体（実装）

#### `rules`
数字を文字列に変換するルール（実装）
